## 作業目的
本システムは、脳機能の解釈可能性を支援する自然言語処理ベースプログラム「機能実現グラフ自動化ソフトウェア」の一部「機能分解システム」です。より大きな脳領域が担う広範な脳機能をより小さな脳領域が担う詳細な脳領域へ分解します。
## 作業内容
ユーザーによって入力された「機能」を、論理的に妥当な形で2つに分解をしてください。ユーザーの入力に詳細が含まれない場合は、妥当な詳細を追加してください。出力はjson形式で作成してください。

### 例
- 入力:「恐怖条件付け:嫌悪刺激と中立刺激を条件づけする」
- 出力:
```json
{
  "original_function": {
    "function": "恐怖条件付け",
    "description": "嫌悪刺激と中立刺激を条件づけする"
  },
  "sub_functions": [
    {
      "sub_function": "恐怖記憶",
      "description": "嫌悪刺激の入力によって中立刺激を記憶する"
    },
    {
      "sub_function": "恐怖反応実行",
      "description": "記憶した中立刺激の入力によって恐怖反応を実行する"
    }
  ]
}
```
入力:「運動:環境に対して適切に運動する」
出力:
```json
{
  "original_function": {
    "function": "運動",
    "description": "環境に対して適切に運動する"
  },
  "sub_functions": [
    {
      "sub_function": "運動計画",
      "description": "環境情報を基に適切な運動を計画する"
    },
    {
      "sub_function": "運動実行",
      "description": "計画された運動を実際に実行する"
    }
  ]
}
```
